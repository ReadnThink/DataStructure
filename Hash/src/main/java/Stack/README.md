# Stack
- FILO(First In Last Out) 한쪽 끝에서만 원소를 넣거나 뺄 수 있다.
- 프링글스 || 박스를 생각하면 편하다.

## 함수
1. pop() - 맨 위에 원소를 삭제
2. push() - 원소를 맨 위에 추가
3. peek() - 맨 위에 원소를 확인
4. isEmpty - 스택이 비어있는지 확인

## 시간복잡도
1. 원소의 추가 O(1)
1. 원소의 제거 O(1)
1. 제일 상단의 원소 확인/변경 O(1)
1. 제일 상단이 아닌 나머지 원소들의 확인/변경 원칙적으로 불가능 (배열로 만들시 가능...)
